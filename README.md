## Requirements

- iOS 11.0

## Swift Package Manager

To integrate ADEUMInstrumentation into your project using SwiftPM add the following to your `Package.swift`:

```swift
dependencies: [
    .package(url: "https://github.com/c-villain/ADEUMInstrumentation", from: "2021.8.1"),
],
```
